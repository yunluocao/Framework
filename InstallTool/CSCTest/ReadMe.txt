This function to test installation and do the sanity testing

1.Go to the install folder,create the Version folder, like CSC630 represents CSC6.3
2.In this folder, create named "server" and "client" folder
3.Unpackage the server and client install files to the related folder
4.Open the installConfig.ini to fill the value according to the parameter explaination
5.Copy the license.dat,installConfig.ini,CSCFull.bat,CSCInstallValidation.py to the install root folder.
6.It will validate the log pass or not
7.Check the service name,register,control panel and start menu(this is for windows)
8.Start up the service,check whether it could be started up.
9.Initial the environment including the product folder engine part
10.Run the engine basic testing
